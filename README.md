## Networking

### Ping Host

`ping` host

### Get whois for domain

`whois` domain.com

### Get DNS for domain

`dig` domain.com

### Reserve loockup host

`dig-x` host

### Outputs the webpage from url

`curl` url

### Connect at host as user

`ssh user@host`

### Change directory permissions

`chown -R www-data:www-data /var/www/example.com`

`chmod 775 /var/www`
